# BookManager_MVVM

Open BookManagement

1. Open Microsoft SQL server run ScriptBook

2. Open BookManagement -> Click folder Model -> database DBBook.dbml

3. Open Server Explorer -> Data Connections -> Add Connection 
3.1 Get Server Name from Microsoft SQL server
3.2 Input Server Name to Add Connection
3.3 Select or enter a Database name -> DBBook
3.4 Add Table and Store Proceduce to DBBook.dbml
    
4. Check connectionStrings at App.config

5. Run Project BookManagement

6. About References
6.1 Add Caliburn.Micro.3.2.0
6.2 Add Caliburn.Micro.Core.3.2.0
6.3 Add iTextSharp.5.5.13.1
6.4 Add Microsoft.Office.Interop.Word.15.0.4797.1003
